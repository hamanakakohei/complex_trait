## 内容
注目しているバリアント群（e.g. eQTL）が、CAUSALdbの高PIP群に濃縮しているかフォレストプロットを描く。
で、バリアントが高PIP群で見られた場合のその形質のGWAS sumstaをダウンロードする。
各dbからsumstaをダウンロードする個々のスクリプトはdl_sumstaフォルダを参照。
最後にlocuscompareで図示するためのインプットファイルを作っている。
